# 퍼블리싱 산출물

## 📁 구조
- `/css/` - 스타일시트
- `/js/` - 자바스크립트
- `/images/` - 이미지
- `/font/` - 폰트 (Pretendard, Fontello)
- `/plugins/` - 라이브러리 (jQuery, jqGrid)
- `*.html` - HTML 파일

## 🎯 개발자 적용 방법
1. 이 폴더 전체를 프로젝트에 복사
2. HTML의 경로는 그대로 사용
3. 백엔드 템플릿 엔진에 맞춰 include 처리

## 📌 주의사항
- Font, Plugin 폴더는 필수
- Symbol 이미지 200개 모두 사용됨
- CSS는 main.css만 import
